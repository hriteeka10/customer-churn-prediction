# 📊 Customer Churn Prediction

## 🚀 Project Overview
This project aims to predict whether a customer will churn (leave the service) using machine learning techniques. It helps businesses identify at-risk customers and take proactive retention actions.

---

## 📂 Dataset
- Telco Customer Churn Dataset
- Contains customer demographics, account details, and service usage

---

## 🧠 Key Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Handling class imbalance using SMOTE
- Multiple ML models:
  - Logistic Regression
  - Random Forest
  - XGBoost
- Model evaluation using Accuracy, Recall, and ROC-AUC

---

## 📊 Results
- Best Model: Logistic Regression
- ROC-AUC Score: ~0.75
- Churn Recall: Improved from ~50% to ~80% using SMOTE

---

## 💡 Key Insights
- Customers with month-to-month contracts are more likely to churn
- Higher monthly charges increase churn probability
- Customers with lower tenure are at higher risk
